========================================

1. Download and install TablePlus
2. Click "Create a new connection"
3. Select "MySQL"
4. Enter:
   Name: Laravel MySQL
   Host: 127.0.0.1
   Port: 3306
   User: test_user
   Password: test_password  
   Database: test_db
5. Click "Test" then "Connect"

========================================
 SAMPLE QUERIES TO TRY
========================================

-- Show all tables
SHOW TABLES;

-- View users
SELECT * FROM users;

-- View products
SELECT * FROM products;

-- Count records
SELECT COUNT(*) FROM users;
SELECT COUNT(*) FROM products;

-- Department stats
SELECT department, 
       COUNT(*) as employees,
       AVG(salary) as avg_salary
FROM users 
GROUP BY department;

========================================
 INSTALL MYSQL CLIENT (OPTIONAL)
========================================

If you want command-line access:

brew install mysql-client
export PATH="/opt/homebrew/opt/mysql-client/bin:$PATH"

Then connect with:
mysql -h 127.0.0.1 -P 3306 -u test_user -ptest_password test_db
